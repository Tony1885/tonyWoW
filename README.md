# Premium Agency Web Project

Ce projet est une application web haute performance avec une esthétique "Premium Agency" (inspirée par Awwwards).

## Stack Technique
- **Framework** : Next.js (App Router) + TypeScript
- **Styling** : Tailwind CSS v4
- **Animations** : Framer Motion & GSAP
- **Smooth Scroll** : Lenis
- **Icônes** : Lucide React

## Caractéristiques
- Dark Mode High-End
- Effets Glassmorphism
- Noise Texture Overlay permanent
- Smooth Scrolling fluide
- Curseur personnalisé dynamique
